# Wearless Studio

패션 셀러가 상품 사진 몇 장으로 AI 상세페이지를 만들고 편집·다운로드하는 스튜디오. 이 문서는 용어집이다 — 데이터 shape는 `documents/common_data_contract.md`, 제품 요구사항은 `documents/PRD.md`.

## Language

### 작업 단위

**프로젝트 (Project)**:
상세페이지 1건을 만드는 작업 단위. 제품 입력부터 에디터 산출물까지의 전 과정과 결과를 묶는 최상위 개념.
_Avoid_: detailPage(엔티티명으로 금지), 작업, 세션

**상세페이지**:
프로젝트의 산출물. 에디터 블록을 세로로 이어 붙인 최종 이미지 문서.
_Avoid_: 디테일 페이지, 랜딩

**보관함 (Library)**:
프로젝트 목록. 카드를 열면 해당 프로젝트의 에디터로 들어간다.

**크레딧**:
생성 행동의 과금 단위. 소모 전에 예상량을 반드시 예고한다.

### 컷과 블록

**컷 종류 (CutType)**:
AI가 생성하는 이미지의 성격 분류. 스타일링컷·호리존컷·제품컷·거울샷 넷이다 (ADR-0003, 거울샷은 ADR-0004).

**스타일링컷**:
모델이 매칭 의류와 함께 일상 맥락에서 착용한 컷.
_Avoid_: 일상컷, daily

**호리존컷**:
무지 호리존 배경에서 핏과 실루엣을 보여주는 착용컷.
_Avoid_: 스튜디오컷, studio

**제품컷**:
모델 없이 제품만 보여주는 컷. 고스트·행거·플랫레이 샷을 가진다.

**아우터 열림 정도**:
아우터 착용컷에서 앞부분을 얼마나 열어 보여줄지 정하는 조건. 전체 열림·부분 열림·전체 닫힘 셋 중 하나다.
_Avoid_: 여밈 상태, 착용 상태, 입는 방식

**거울샷**:
거울 앞 셀피 느낌의 리얼 착용컷. 방향 개념이 없고 샷은 전신·무릎만, 얼굴 기본값은 '폰으로 가림', 포즈는 셀피 구도로 자동 고정. 거울 장소는 설정이 아니라 생성예시가 담는다 (ADR-0004).
_Avoid_: 셀피컷, OOTD컷

**내 이미지**:
사용자가 직접 업로드해 그대로 쓰는 이미지. 컷 종류가 아니라 출처(source)의 구분이다.

**블록**:
상세페이지를 구성하는 세로 적층 단위. 콘티보드 블록(생성 계획)과 에디터 블록(편집 대상)이 있다.

**블록 종류 (BlockKind)**:
블록이 상세페이지에서 맡는 섹션 역할(후킹·셀링포인트·스타일링·호리존·제품·정보). 컷 종류와 직교한다.

**자동 블록**:
생성 완료 시 자동으로 덧붙는 안내 블록 — 사이즈 안내, 세탁 안내, AI 생성 안내.

**현재 컷 변형 (Vary)**:
선택한 이미지를 기준으로 방향·샷·포즈·표정·배경을 바꾼 새 컷을 만드는 에디터 기능. 변경 0개로 실행하면 '비슷한 컷 만들기'다.

### 생성 플로우

**분석 (Analysis)**:
AI가 제안하고 사용자가 확인·수정하는 상품 속성 묶음(세부 카테고리·성별·핏·소재·강조 특징·모델·매칭 의류).

**실측**:
사용자가 직접 입력하는 cm 단위 치수. AI는 절대 추정하지 않는다. 상품 정보(Product)의 일부다.

**강조 특징 (Selling Point)**:
카피와 이미지 생성 방향에 쓰이는 핵심 셀링 포인트. 최대 5개.

**마네킹컷**:
실제 이미지 생성 전 핏·실루엣 기준을 잡는 마네킹 착용 이미지. 후보 A/B와 버전 히스토리를 가진다.

**조정 (Adjust)**:
선택한 마네킹컷의 총기장·핏을 바꾸는 행동. 프로젝트당 2회 제한이며 전부 재생성도 횟수를 소모한다.

**매칭 의류**:
스타일링컷에서 함께 입힐 의류. 첫 번째 선택이 메인, 두 번째가 서브.

**콘티보드**:
생성 전에 컷 구성·순서·조건을 정하는 계획 보드. 썸네일은 방향을 잡기 위한 예시이며 최종 이미지가 아니다.
_Avoid_: 초안 페이지

**구성 방식 (ComposeMode)**:
콘티보드의 초기 컷 구성 프리셋 — 간단형·기본형·확장형.

**모델**:
AI 생성 착용컷에 등장하는 가상 인물 프리셋.

### 색상과 이미지

**기준 색상**:
의류 구조와 디테일 분석의 대표 색상 그룹(색상 1). 앞면 이미지가 필수다.
_Avoid_: 베이스 컬러

**추가 색상**:
색상 옵션 표현용 색상 그룹. 최대 3개, 정면 사진 필수.

**스와치 (Swatch)**:
12색 고정 팔레트의 색상 식별자. 색상 그룹의 정체성은 스와치로 정해지며, 고르기 전에는 '색상 미정'이다.

**모노톤**:
스와치 중 화이트·그레이·블랙·아이보리·베이지.

**의류 탭 (Wardrobe)**:
에디터에서 캔버스에 넣을 수 있는, 색상 그룹별로 묶인 이미지 라이브러리.
_Avoid_: 갤러리
