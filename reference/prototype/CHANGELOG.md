# Wearless 프론트엔드 — PRD 대비 변경점 (최종)

> **목적.** 현재 구현된 프론트엔드를 기준으로, `wearless_prd_integrated_v2.md`(PRD 통합본) 대비 **달라진 점**을 PRD 섹션 순서로 정리한다. PRD를 최종 갱신할 때 이 문서를 반영하면 된다.
> **관점.** "레이아웃 시안 탐색"·"Tweaks 전환"·"실험 후 롤백" 같은 과정 기록은 의도적으로 제외하고, **확정된 구현 결과만** 적는다.
> 최종 갱신: 2026-06-09

---

## 0. 구현 환경 (참고)

- **현재 산출물은 빌드 없는 프로토타입**이다 — 단일 HTML + CDN React + 브라우저 Babel. 실제 서비스 구현 시 Next.js + 번들러로 이관하며, 손으로 구현한 캔버스 드래그/리사이즈/레이어 등은 표준 npm(`react-moveable`·`react-easy-crop` 등)으로 대체될 수 있다.
- 레이아웃은 **단일 확정안으로 고정**되어 있다. (탐색용 레이아웃 변형과 Tweaks 패널은 제거됨 — PRD 기능 아님)
- 파일 구조: `features/`(화면별), `components/`(공용), `styles/`(tokens·app·features), `mock/`(가짜 데이터·크레딧 단가 설정).

## 공통 — 비주얼·모션 (디자인 시스템 적용 결과)

- 선택 칩 = 흰 배경 + Sky(#8fbfee) 테두리, 비선택 = 살짝 옅은 회색.
- 세그먼트(흰 pill 슬라이딩) · 언더라인 탭(밑줄 슬라이딩).
- 강조 특징·AI 편집 pill·토스트 테두리에 **glow 4색 conic**(약하게 순환).
- 토스트: 우측 상단·흰 배경·페이드아웃.
- 진행바: 완료=검정, 현재=포인트(#4f88c9), 연결선 완료 구간 검정.
- 포인트 컬러(#4f88c9 / Sky)는 **강조 용도로만**(선택·링크·활성), CTA는 검정.
