# Wearless PRD — 간소화본 (구현 확정)

> AI가 상품 사진·정보로 쇼핑몰 상세페이지를 자동 생성·편집해주는 서비스.
> 이 문서는 **무엇을 만드는가**만 빠르게 본다. 데이터 모델·API·인터랙션 명세는 **상세본**(`PRD_상세본.md`) 참고.
> 최종 갱신: 2026-06-09

---

## 1. 한 줄 요약

상품 사진 몇 장 + 기본 정보를 넣으면 → AI가 분석하고 → 마네킹 착용컷을 만들고 → 상세페이지 초안(콘티)을 짜고 → 한 번에 생성한 뒤 → 셀러가 에디터에서 다듬어 → PNG로 내보낸다.

## 2. 전체 플로우 (4단계)

```
[제품 정보 + AI 분석]  →  [마네킹컷]  →  [콘티보드]  →  [에디터]
       1단계                2단계         3단계         4단계
   (입력 후 같은 화면         A/B 후보      블록 구성·    생성된 상세페이지
    아래로 분석 전개)        선택·조정      편집          직접 수정 → 다운로드
```

- 스텝퍼는 **4개 점**으로 표기. 생성 대기 화면은 별도 단계가 아니라 콘티→에디터 사이의 로딩.
- 상단 네비: 로고 · 상세페이지 제작 · 보관함 · 크레딧 잔액 · 플랜 · 프로필.

## 3. 화면별 핵심

### ① 제품 정보 입력 + AI 분석 (한 화면)
- **입력은 상품명 + 상품 이미지만.** (상품명은 선택 — 비우면 AI가 지어줌)
- 이미지: **앞면(필수)·뒷면·디테일·착용** 4각도 + **색상별**(최대 3색) 업로드. 드래그앤드롭/클릭, 파일명·용량·형식 표시.
- 앞면 1장 이상이어야 "입력 완료" 활성.
- "입력 완료" → **같은 화면 아래로 AI 분석 결과가 인라인 전개**(입력부는 요약 카드로 접힘).

### ② AI 분석 결과 (위 화면에 이어짐)
하나의 카드 안에 순서대로:
- **기본 정보** — 의류 종류·세부 카테고리·성별·핏 (칩, 선택 해제 가능)
- **소재** — 소재명+% 칩 (합계 100% 권장, 초과 시 경고)
- **실측 정보** — 총장·어깨너비·가슴단면·소매길이 **빈칸 직접 입력**, `☐ 실측 모름`
- **강조하고 싶은 특징** — 칩, AI 추천 자동 채움, **최대 5개** (`N/5개` 표시)
- **모델** — 카드 선택
- **매칭 의류** — 최대 2개(메인/서브)

### ③ 마네킹컷
- **A/B 2후보** + 히스토리. 하나 선택.
- **세부 조정** — 총기장 → 핏 (가운데="현재" 기준 세그). 메인 의류 + 매칭 의류(토글). **조정 2회 제한.**
- **상세페이지 구성 방식**(간단형/기본형/확장형)은 우측 미니 카드로 선택, 각 예상 컷 수 표기.
- "마네킹컷 전부 재생성"은 확인 모달.

### ④ 콘티보드 (상세페이지 초안)
- 블록 카드 목록(좌) + 선택 시 인스펙터(우) 분할.
- 카드 = **컷 종류 + 썸네일**(호버 시 방향·샷·색상). 드래그로 순서 변경.
- 컷 종류: **호리존컷 / 일상컷 / 제품컷** (제품컷은 방향=앞/뒤, 샷=고스트/행거/플랫레이).
- 인스펙터: 방향·샷·색상(원)·**분위기 예시**(컷·방향·샷에 따라 갱신)·포즈·매칭 의류·모델 얼굴·앵글.
- 내 이미지 업로드 블록 + 블록 사이 삽입.
- 하단: 카피라이팅 ON/OFF · "이대로 생성하기 → {N} 크레딧".

### ⑤ 생성 대기
- 단계별 진행 표시 후 에디터 진입.

### ⑥ 에디터 (유료 핵심)
- 좌측 도구: **AI · 의류 · 이미지 · 프레임 · 텍스트 · 오브젝트 · 레이어**
- 중앙: 1000px 캔버스(블록 세로 나열, 줌). 우측: 미니 프리뷰(드래그 재정렬).
- 요소 조작: **드래그 이동 · 리사이즈 · 회전(스냅) · 인라인 텍스트 · 다중 선택 · 레이어**.
- 블록: 크기 고정(넘치면 클립), 상/하 핸들로 높이 조절, 배경색.
- 생성 끝에 **사이즈·세탁·AI 생성 안내 블록 자동 추가.**
- 단축키: 줌(Ctrl+휠) · T · Undo/Redo · Ctrl+S · `[`/`]`(레이어).
- 미리보기(전 블록 합본) · 저장 · **다운로드**(긴 PNG / 블록별 ZIP).

### ⑦ 보관함
- 상세페이지 카드(완료/생성 중/초안 상태) + 새로 만들기.

## 4. 크레딧

- 작업 전에 **예상 차감을 미리 표기**(마네킹 생성·조정, 콘티 생성, 에디터 이미지 생성).
- 단가는 **미확정** → 한 곳(설정값)에서 관리. 상세본 참고.

## 5. 디자인 원칙 (요약)

- **Wearless 디자인 시스템**(= Cal.com 모노크롬 + 배경 글로우 1종) 준수.
- 순백 배경, 깊이는 그림자로만, 포인트 컬러(Sky/#4f88c9)는 선택·링크·활성에만 작게. CTA는 검정.
- 폰트: 제목 Cal Sans / 본문·UI Pretendard. 아이콘 Lucide(무채색). 이모지 없음.
